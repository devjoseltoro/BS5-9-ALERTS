Alertas

Las alertas se crean con la clase .alert, seguida de una clase de color contextual:

¡Éxito! Este cuadro de alerta podría indicar una acción exitosa o positiva.
¡Información! Este cuadro de alerta podría indicar un cambio o acción informativo neutral.
¡Advertencia! Este cuadro de alerta podría indicar una advertencia que podría necesitar atención.
¡Peligro! Este cuadro de alerta podría indicar una acción peligrosa o potencialmente negativa.
¡Primario! Indica una acción importante.
¡Secundario! Indica una acción un poco menos importante.
¡Oscuro! Alerta gris oscuro.
¡Ligero! Alerta gris claro.

Enlaces de alerta

Agregue la clase .alert-link a cualquier enlace dentro del cuadro de alerta para crear "enlaces de colores coincidentes":

Alertas de cierre

El botón con class="btn-close" y data-bs-dismiss="alert" se usa para cerrar el cuadro de alerta.

La clase alert-dismissible alinea el botón a la derecha.
